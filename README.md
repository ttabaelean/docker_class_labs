# 🐳 Docker Infrastructure Lab
- 이 저장소는 도커 컨테이너 기술 및 오케스트레이션 실습을 위한 예제와 가이드라인을 보관하는 공간입니다. 
- 기초적인 컨테이너 사용법부터 실무 기반의 서비스 배포까지 단계별 자료가 업데이트될 예정입니다.
- 처음 도커 컨테이너 학습자에게 적합하게 만들어졌습니다.

---

### 🏗️ 실습 개요
- 현대적인 클라우드 네이티브 환경의 핵심인 도커(Docker)를 활용하여 애플리케이션을 효율적으로 컨테이너화하고 관리하는 과정을 학습합니다.
- 운영 환경: Rocky Linux 9 / Ubuntu 기반 Docker Engine
- 학습 목표: 이미지 빌드, 컨테이너 운영, 데이터 영속성 관리 및 네트워크 구성 역량 습득
